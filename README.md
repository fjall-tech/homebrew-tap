# Fjall Homebrew Tap

Homebrew formulae for [Fjall](https://fjall.dev) CLI tools.

## Installation

```sh
brew install fjall/tap/fjall
```

Or tap first, then install:

```sh
brew tap fjall/tap
brew install fjall
```

## Upgrading

```sh
brew upgrade fjall
```

## Uninstalling

```sh
brew uninstall fjall
```

## Alternative: npm

```sh
npm install -g @fjall/cli
```
